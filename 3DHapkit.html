Three-Degree-Of-Freedom Device

Students at the Johns Hopkins University have designed a three-degree-of-freedom (3-DOF) device based on the Hapkit and delta-mechanism. The system can be built using commonly available bearings and screws as well as laser-cut and 3D-printed structural components. The demo currently only renders a flat surface with two holes (right image). Future work will include the implementation of the Jacobian-transpose for force control as well as an assembly video.

The 3D Hapkit was presented as a demo and WIP paper at the 2023 IEEE World Haptics Conference.

    Install the Leader code on the Arduino that's connected to the PC and the Follower code on the two other Arduinos.
    On the PC, install Processing and load the deltaProcessing application. Power all three boards and connect the Leader Arduino to the PC. Ensure that the COM port is correct by checking the COM port of the Arduino in Device Manager. Once everything is running you will see the green dot track the movement of the 3D hapkit and be able to interact with the virtual surface.
